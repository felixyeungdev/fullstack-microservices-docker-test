# Example Microservices with Docker

## Folder Structure
```
.
├── backend
│   └── services
│       ├── orders
│       ├── products
│       ├── restaurants
│       ├── shops
│       ├── transactions
│       └── users
├── frontend
│   └── web
└── gateway
```

## Run
```sh
docker compose build
docker compose up
```

Visit http://localhost:80/
